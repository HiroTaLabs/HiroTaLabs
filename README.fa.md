## 👨‍💻 درباره من

من خالق **HiroTaLabs** هستم؛ تمرکز من روی ساخت سیستم‌های نرم‌افزاری واقعی و کاربردی با علاقه‌ای ویژه به توسعه فول‌استک، معماری بک‌اند و برنامه‌های وب مقیاس‌پذیر است. در حال حاضر به بررسی الگوهای مدرن مهندسی نرم‌افزار و توسعه معماری‌های Decoupled (اتصال فرانت‌اندهای پویا به ای‌پ‌آی‌های قدرتمند) می‌پردازم. همچنین دوست دارم تکنولوژی را با فلسفه و تفکر خلاق ترکیب کنم تا تجربه‌های دیجیتال معناداری بسازم.

## 🎯 تمرکز فعلی

* توسعه وب فول‌استک (اتصال فرانت‌اند React به بک‌اند Django از طریق REST API)
* معماری سیستم، دیتابیس‌های رابطه‌ای (MySQL / SQL) و طراحی API
* ساخت پروژه‌های واقعی و آماده‌ی پروداکشن همراه با پنل‌های مدیریتی و جریان‌های محصولی کاربردی
* استقرار (Deployment)، کانتینرسازی (Docker) و محیط‌های لینوکس (Linux)
* توسعه سیستم‌های تعاملی ربات تلگرام
